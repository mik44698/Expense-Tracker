# Expense Tracker 

A clean, beginner-friendly **Flask expense tracking web application** that allows users to record, view, and delete expenses with persistent storage using SQLite.
## Features
- Add expenses with amount, category, and optional notes
- View recent expenses in a clean dashboard
- Automatically calculate total spending
- Delete expenses with a single click
- Data persisted using SQLite
## Tech Stack
- **Python**
- **Flask**
- **Flask-SQLAlchemy**
- **SQLite**
- **HTML / Bootstrap**
